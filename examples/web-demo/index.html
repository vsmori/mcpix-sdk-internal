Por que esta demo demonstra o protocolo

Substituição institucional: o C₂ gerado pelo recebedor offline (retained_c2) é bit-exato igual ao C₂ reconstruído pelo pagador (recovered_c2) a partir do mesmo (Seed, T, C₁). O recebedor nunca transmitiu C₂ — só o pagador o derivou com a Seed que veio do banco recebedor.

Anti-replay: tentar validar o mesmo C₂ duas vezes devolve Replay — o store local marcou o registro como consumido depois da primeira validação.

Anti-tampering: qualquer bit alterado no campo de transporte produz C₁ diferente, que com a mesma Seed/T deriva outro C₂ — Mismatch na validação.

Tamanho: o wasm bundle inteiro tem ~80 KB. A SDK Rust roda no browser sem polyfills.
